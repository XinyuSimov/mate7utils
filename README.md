### 框架介绍

## 使用基础
* 针对电信定制机
* Root权限(EMUI4.0.1解锁后Root)
* ProjectMenuAct.apk(APK/ProjectMenuAct.apk)
	需自行安装
* 修改/data/cust/prop/local.prop
	ro.config.hw_opta=92改01
	
	
## 网络切换
* 打开联通网络
		支持联通、移动4G，联通3G，联通移动2G

* 恢复电信网络
	恢复Mate7电信定制机原网络设置
